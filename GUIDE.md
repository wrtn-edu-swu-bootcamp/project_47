# 판교어 번역기 - 사용 가이드

## 📱 시니어를 위한 사용법 (큰 글씨)

### 1단계: 웹사이트 접속하기
1. 인터넷 브라우저(크롬, 엣지 등)를 엽니다
2. 주소창에 웹사이트 주소를 입력합니다
3. 엔터키를 누릅니다

### 2단계: 모르는 단어 찾기
1. 화면 위쪽의 **검색창**을 찾습니다
2. 모르는 단어를 입력합니다
3. **검색 버튼**을 누릅니다

### 3단계: 설명 듣기
1. 검색 결과가 나타납니다
2. 🔊 **스피커 모양 버튼**을 누릅니다
3. 소리로 설명을 들을 수 있습니다

### 음성으로 검색하기
1. 검색창 옆의 **🎤 마이크 버튼**을 누릅니다
2. "듣고 있습니다"라는 글씨가 나타나면 말합니다
3. 검색하고 싶은 단어를 또박또박 말합니다
4. 자동으로 검색됩니다

## 💡 MZ 세대를 위한 꿀팁

### 빠른 검색
- **Ctrl + F** 또는 헤더의 검색창 사용
- 자동완성 지원

### 실시간 번역
1. 웹페이지의 판교어를 **마우스로 드래그**
2. 자동으로 번역 팝업 등장
3. 클릭 없이 바로 확인 가능

### 트렌드 워드
- "요즘 그거 뭔데?" 카테고리 선택
- 최신 밈과 트렌드 용어 큐레이션

## 🔧 브라우저 확장 프로그램 설치

### Chrome/Edge
1. `chrome://extensions/` 접속
2. "개발자 모드" ON
3. "압축해제된 확장 프로그램 로드"
4. `extension` 폴더 선택
5. 완료!

### 사용법
- 웹페이지에서 단어 드래그 → 자동 번역
- 확장 프로그램 아이콘 클릭 → 수동 검색

## ❓ 자주 묻는 질문

### Q1. 음성 검색이 안 돼요
**A.** 브라우저에서 마이크 권한을 허용해야 합니다.
1. 브라우저 주소창 왼쪽의 🔒 자물쇠 클릭
2. "마이크" 권한을 "허용"으로 변경

### Q2. 내가 찾는 단어가 없어요
**A.** 비슷한 단어로 검색해보세요. 계속 없다면 피드백을 남겨주세요!

### Q3. 텍스트 드래그 번역이 안 돼요
**A.** 사전에 등록된 용어만 번역됩니다. 검색 페이지에서 직접 검색해보세요.

### Q4. 모바일에서도 사용할 수 있나요?
**A.** 네! 웹사이트는 모바일에서도 사용 가능합니다. (확장 프로그램은 PC 전용)
