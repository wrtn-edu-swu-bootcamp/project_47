## 🚀 실행 방법

### 웹 애플리케이션
```bash
cd c:\Users\PC\Desktop\0123_test
npm install    # 이미 완료
npm run dev    # 현재 실행 중!
```

**접속:** http://localhost:3000

### 브라우저 확장 프로그램
1. Chrome에서 `chrome://extensions/` 접속
2. "개발자 모드" 활성화
3. "압축해제된 확장 프로그램 로드"
4. `c:\Users\PC\Desktop\0123_test\extension` 폴더 선택
